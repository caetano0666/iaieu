Como publicar

  1. Preencha o formulário e clique em Gerar posts.json
  2. Clique em Baixar posts.json
  3. Se tiver imagem, suba a imagem também no GitHub
  4. Acesse github.com/caetano0666/iaieu
  5. Clique em + → Upload files
  6. Sobe o posts.json (e a imagem se tiver)
  7. Clique em Commit changes — pronto!
